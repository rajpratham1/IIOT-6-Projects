# Online Store WebApp

A modern, responsive online store website built using Flask for the IIOT-6 WebApp Suite (Invertis University, 6th Semester, B.Tech CSE).

## Features
- Home, Shop, Cart, Checkout, Contact pages
- Bootstrap-based UI with animations and effects
- Organized static and template folders
- Ready for backend/database integration (Phase 2)
- AI tools integration planned (Phase 3)

## Technologies Used
- Python (Flask)
- HTML, CSS, JavaScript
- Bootstrap

## Folder Structure
```
OnlineStoreWebApp/
├── app.py
├── requirements.txt
├── static/
│   ├── css/
│   ├── img/
│   ├── js/
│   └── lib/
├── templates/
│   ├── index.html
│   ├── shop.html
│   ├── cart.html
│   ├── checkout.html
│   └── contact.html
```

## Deployment
- Ready for deployment on Render.com
- Use `requirements.txt` for dependencies
- Start command: `python app.py`
